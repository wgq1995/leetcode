# 题目描述
    在 O(n log n) 时间复杂度和常数级空间复杂度下，对链表进行排序。 
# 示例
```
输入: 4->2->1->3
输出: 1->2->3->4

输入: -1->5->3->4->0
输出: -1->0->3->4->5
```
# 思路
   归并排序
   １.快慢指针找中点
   ２.分割为两个mergesort，basecase是只有一个节点的情况
   ３.merge两个排序好的列表
# 代码
```
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None
class Solution:
    def sortList(self, head: ListNode) -> ListNode:
        if not head or not head.next:
            return head
        p1, p2 = head, head
        while p2.next and p2.next.next:
            p1 = p1.next
            p2 = p2.next.next
        p3 = p1.next
        p1.next = None
        head = self.sortList(head)
        p3 = self.sortList(p3)
        return self.merge(head, p3)

    def merge(self, head1, head2):
        p = ListNode(0)
        h = p
        while head1 and head2:
            if head1.val < head2.val:
                p.next = head1
                head1 = head1.next
            else:
                p.next = head2
                head2 = head2.next
            p = p.next
        if head1:
            p.next = head1
        else:
            p.next = head2
        return h.next
```
